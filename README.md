# xey_invest
Xëy Invest est une plateforme de financement participatif inspirée du modèle de la tontine sénégalaise ("nateu"). Elle permet à des jeunes porteurs de projets de lever des fonds auprès d'une communauté d'investisseurs, en échange de parts dans leur entreprise. L'objectif est de démocratiser l'investissement  pour financer l'innovation locale
